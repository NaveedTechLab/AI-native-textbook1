# Hardware Setup Skill

## Purpose
This skill enables the configuration and setup of physical hardware for the Physical AI & Humanoid Robotics textbook. It provides specialized knowledge for setting up robotic platforms, compute systems, sensors, and peripherals.

## Capabilities
- Configure RTX workstation setups for AI/robotics development
- Set up NVIDIA Jetson platforms for edge computing
- Configure Unitree quadruped robots and other robotic platforms
- Integrate sensors (cameras, lidars, IMUs, etc.) with compute platforms
- Configure networking and communication protocols
- Set up real-time operating systems for robotics applications
- Provide hardware troubleshooting and diagnostics
- Recommend hardware specifications for different use cases

## Usage Context
- Invoked when users need hardware setup assistance
- Used for configuring robotic platforms for exercises
- Applied to capstone project hardware requirements
- Integrated with the AI textbook assistant

## Input Format
- Target hardware platform (RTX workstation/Jetson/Unitree/etc.)
- Specific configuration requirements
- Integration needs with software components
- Performance and real-time requirements

## Output Format
- Step-by-step hardware setup instructions
- Driver and firmware installation procedures
- Configuration files and parameters
- Performance optimization recommendations
- Safety and troubleshooting guidelines
- Hardware compatibility notes

## Constraints
- Must follow manufacturer's safety guidelines
- Should consider power and thermal requirements
- Must account for real-time performance needs
- Should provide cost-effective solutions
- Must be suitable for educational purposes
