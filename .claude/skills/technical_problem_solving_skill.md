# Technical Problem Solving Skill

## Purpose
This skill enables systematic problem-solving for technical issues in the Physical AI & Humanoid Robotics textbook. It provides structured approaches to diagnose and resolve problems across software, simulation, and hardware domains.

## Capabilities
- Diagnose complex robotics system issues
- Troubleshoot ROS 2 runtime problems
- Solve simulation environment issues
- Debug hardware configuration problems
- Identify performance bottlenecks
- Resolve networking and communication issues
- Apply systematic debugging methodologies
- Provide root cause analysis

## Usage Context
- Invoked when users encounter technical problems
- Used for debugging complex multi-component systems
- Applied to troubleshooting textbook exercises
- Integrated with the AI textbook assistant

## Input Format
- Problem description and symptoms
- System configuration and environment
- Error messages and logs
- Steps already taken to resolve the issue

## Output Format
- Systematic problem diagnosis
- Root cause analysis
- Step-by-step resolution procedures
- Alternative solution approaches
- Prevention recommendations
- Diagnostic commands and tools

## Constraints
- Must follow systematic troubleshooting methodologies
- Should prioritize safety in hardware troubleshooting
- Must consider system stability
- Should provide verified and tested solutions
- Must be suitable for educational purposes
