# Implementation Assistance Skill

## Purpose
This skill provides comprehensive assistance for implementing robotics systems from the Physical AI & Humanoid Robotics textbook. It offers architectural advice, best practices, and implementation strategies for complex robotics applications.

## Capabilities
- Design system architectures for robotics applications
- Provide best practices for robotics software development
- Recommend appropriate algorithms for specific tasks
- Guide integration of multiple components
- Suggest performance optimization strategies
- Advise on real-time system design
- Recommend testing strategies for robotics systems
- Provide project planning and execution guidance

## Usage Context
- Invoked when users need implementation guidance
- Used for complex system design decisions
- Applied to capstone project implementations
- Integrated with the AI textbook assistant

## Input Format
- System requirements and constraints
- Target platform and performance requirements
- Integration needs with existing components
- Specific implementation challenges or questions

## Output Format
- Architectural recommendations and patterns
- Implementation roadmap and timeline
- Best practice guidelines and standards
- Performance optimization suggestions
- Risk mitigation strategies
- Testing and validation approaches
- Code structure and organization advice

## Constraints
- Must consider real-time performance requirements
- Should follow robotics software engineering best practices
- Must account for safety and reliability
- Should provide scalable and maintainable solutions
- Must be suitable for educational purposes
