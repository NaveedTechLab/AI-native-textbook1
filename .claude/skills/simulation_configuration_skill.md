# Simulation Configuration Skill

## Purpose
This skill enables the configuration of simulation environments for robotics applications in the Physical AI & Humanoid Robotics textbook. It specializes in setting up Gazebo, Unity, and NVIDIA Isaac simulations.

## Capabilities
- Configure Gazebo simulation environments
- Set up Unity robotics simulations
- Create NVIDIA Isaac simulation scenarios
- Design robot models and URDF files
- Configure sensor models and parameters
- Set up physics properties and constraints
- Create complex simulation scenarios
- Optimize simulation performance

## Usage Context
- Invoked when users need simulation setup help
- Used for creating virtual testing environments
- Applied to textbook examples and exercises
- Integrated with the AI textbook assistant

## Input Format
- Target simulation platform (Gazebo/Unity/NVIDIA Isaac)
- Robot model requirements
- Sensor configuration needs
- Environment parameters
- Performance requirements

## Output Format
- Complete simulation configuration files
- Robot model definitions (URDF/SDF)
- Sensor configuration files
- Launch scripts and setup instructions
- Performance optimization recommendations
- Troubleshooting guides

## Constraints
- Must be compatible with target simulation platform versions
- Should provide realistic physics parameters
- Must consider computational performance
- Should follow simulation best practices
- Must be suitable for educational purposes
